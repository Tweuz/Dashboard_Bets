📊 Dashboard Analítico: O Impacto Socioeconómico das Bets no Brasil
Este projeto é uma plataforma web de análise exploratória focada no cruzamento de dados sobre transações via Pix destinadas a plataformas de apostas (bets), indicadores de incumprimento familiar (inadimplência) e a leitura socioeconómica do território brasileiro.

Foi desenvolvido com um forte foco em Acessibilidade Web, seguindo boas práticas alinhadas ao eMAG, modelo do Governo Digital que orienta o desenvolvimento de páginas, portais e serviços acessíveis para o maior número de pessoas possível 
.

🎯 Objetivo do Projeto
Estruturado para apoiar a visualização de dados e a interpretação de risco, o painel funciona como uma ferramenta de apoio para investigadores, banca, gestores públicos e entidades de crédito.

O projeto contribui para discussões sobre:

O comprometimento do rendimento básico familiar.

A ludopatia digital.

A pressão financeira em famílias vulneráveis.

A relação entre comportamento de consumo, vulnerabilidade social e apostas online.

✨ Funcionalidades Principais
Visualização de dados interativa com gráficos, indicadores e elementos analíticos em dashboard.

Filtros exploratórios por período, região e variáveis relevantes para leitura dos dados.

Design responsivo para computadores, tablets e smartphones.

Exibição de métricas de impacto para facilitar a leitura dos principais indicadores.

Estrutura preparada para expansão com novos recortes estatísticos e melhorias visuais.

♿ Foco em Acessibilidade (A11y)
A acessibilidade é um dos pilares deste projeto. O desenvolvimento considera práticas que tornam a navegação mais inclusiva e funcional para diferentes perfis de utilizadores.

Recursos e preocupações de acessibilidade aplicados no projeto:

Estrutura semântica com HTML5 e uso de atributos ARIA para melhorar a interpretação por tecnologias assistivas.

Referência ao eMAG, conjunto de recomendações do Governo Digital para tornar páginas e serviços mais acessíveis 
.

Possibilidade de integração com o VLibras, suíte de ferramentas do Governo Brasileiro voltada à tradução de conteúdos digitais para Libras 
.

Leitura de ecrã por voz com suporte à API window.speechSynthesis, interface da Web Speech API usada para síntese de fala no navegador 
.

Navegação por teclado, contraste adequado e adaptação visual como parte da preocupação com inclusão digital.

🛠️ Tecnologias Utilizadas
O projeto foi desenvolvido com foco em leveza, simplicidade e desempenho, evitando dependências pesadas.

HTML5: Estrutura semântica e apoio à acessibilidade.

CSS3: Estilização, responsividade, organização visual e adaptação para diferentes ecrãs.

JavaScript (Vanilla): Interatividade, manipulação de DOM, lógica do dashboard e recursos de acessibilidade.

Web Speech API: Base para funcionalidades de síntese de voz em navegadores compatíveis 
.
